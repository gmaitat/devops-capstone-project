# devops-capstone-project

Proyecto de microservicio de cuentas de clientes para plataforma de e-commerce.

## Descripción
Este repositorio contiene la planificación ágil para el desarrollo de un microservicio de cuentas que permitirá gestionar información básica de clientes (nombres y direcciones) a través de una API REST.

## Objetivo
Implementar un servicio de cuentas con operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y listar clientes, siguiendo prácticas de desarrollo ágil con GitHub Projects.
